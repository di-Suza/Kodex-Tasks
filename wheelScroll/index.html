How it works

The Logic

01

wheel event

Browser window pe wheel event listen karte hain. Ye tab fire hota hai jab user mouse scroll karta hai.

02

deltaY value

e.deltaY positive hoti hai jab neeche scroll ho, negative hoti hai jab upar scroll ho.

03

translateY trick

Navbar ko translateY(-100%) se hide karte hain. CSS transition se smooth animation milti hai.

04

class toggle

Sirf ek class .hidden add/remove karte hain JS se. Baaki sab CSS handle karta hai.

The Code

Under the
hood

// Wheel event listen karo
window.addEventListener('wheel', (e) => {
  // deltaY > 0 = neeche ja raha → hide
  if (e.deltaY > 0) {
    navbar.classList.add('hidden');
  } else {
    // deltaY < 0 = upar ja raha → show
    navbar.classList.remove('hidden');
  }
});
deltaY > 0

Neeche scroll

Navbar hide — .hidden class add → translateY(-100%)

deltaY < 0

Upar scroll

Navbar show — .hidden class remove → translateY(0)
